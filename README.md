✨ Portfolio React 2026

Portfolio personnel développé avec React pour présenter mes projets, mes compétences et mon univers de développement front-end.
Le site suit une approche minimaliste et moderne, avec une navigation simple et une attention particulière portée au responsive et à la lisibilité.


🚧 Projet en cours de développement:

Les textes définitifs ne sont pas encore intégrés
Tous les projets ne sont pas encore ajoutés
La police finale n’est pas encore choisie
Le logo n’est pas encore créé
Certaines parties du design peuvent encore évoluer
Les animations ne sont pas définitives
Amélioration des performances


⚙️ Stack utilisée:

⚛️ React
⚡ Vite
🎨 SCSS
🎬 Framer Motion
🧭 React Router
📂 Données locales (JSON)
🧶 Yarn


🧩 Fonctionnalités:

Galerie horizontale sur la page d’accueil
Page avec tous les projets (layout en grid)
Page About
Page dynamique pour chaque projet
Navigation fluide avec React Router
Design minimaliste
Responsive mobile / tablette / desktop
Structure de composants claire et réutilisable


📁 Structure du site:

/ → Page d’accueil
/projects → Tous les projets
/projects/:id → Page projet dynamique
/about → Présentation


🎯 Objectif du projet:

Présenter mes projets récents
Améliorer ma maîtrise de React
Travailler la structure d’un projet front-end complet
Améliorer le responsive design
Créer une interface moderne et minimaliste
